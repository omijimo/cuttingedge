# Data Directory

## Structure

- `sample_input/` — Sample MIDI files for testing inference
- `sample_output/` — Example output MIDI files produced by the pipeline
- `processed/` — Preprocessed dataset (generated by `preprocess` command, git-ignored)

## Generating Synthetic Data

```bash
python -m accompaniment.cli preprocess --config configs/base.yaml
```

This creates a synthetic dataset of melody-chord pairs in `data/processed/`.

## Using Real Datasets

The preprocessing pipeline supports adapters for real datasets. See `src/accompaniment/data/preprocess.py` for the `MidiChordAdapter` class.

### POP909-style datasets

Place MIDI files and chord annotations in a directory, then use the adapter:

```python
from accompaniment.data.preprocess import MidiChordAdapter
adapter = MidiChordAdapter(midi_dir="path/to/midis", chord_dir="path/to/chords")
```

### Chord annotation format

Simple text files with one chord per line: `<time_in_seconds> <chord_label>`

```
0.0 Cmaj
2.0 Fmaj
4.0 Gmaj
6.0 Cmaj
```
